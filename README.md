# AI DeepSearch Agent

An intelligent, multi-step research assistant built with LangChain and LangGraph.
This AI agent automates deep information gathering by scraping the web, compiling historical insights, and generating a structured PDF report with timestamps.
Designed for students, researchers, and personal study, the agent helps simplify complex topics and produce well-organized, ready-to-reference research documents.
